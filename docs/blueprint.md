# **App Name**: Hafash.pk

## Core Features:

- Luxury Multi-Role Access: Secure authentication for photographers and clients with distinct, polished dashboard experiences tailored for production-quality workflow.
- Event & Gallery Orchestration: Comprehensive system to create and manage cinematic event metadata, storage metrics, and localized unique gallery links, all stored within the database.
- Dynamic Media Pipeline: A sophisticated upload interface featuring drag-and-drop capability, real-time progress indicators, and precise cover photo selection for luxury assets.
- AI Cinematic Highlight Tool: An AI tool that analyzes uploaded sets to suggest and highlight high-quality visual moments for featured highlight reels.
- Intelligent Protection System: Toggleable download protection that dynamically overlays custom watermarks on preview images until downloads are explicitly unlocked.
- Client Curation Hub: Mobile-optimized PWA gallery where clients can select favorites using a heart system, syncing immediately to the photographer’s curation panel.
- Expressive Delivery Module: Instant delivery suite allowing one-click sharing via WhatsApp and secure public link copying to fulfill the promise of delivering memories beautifully.

## Style Guidelines:

- The palette is centered on a Royal Gold primary color (#D4AF37) to evoke high-end craftsmanship and wedding prestige.
- The background uses a deep Obsidian Charcoal (#1A1916), maintaining a desaturated golden undertone for a sophisticated dark-mode atmosphere.
- An accent of Deep Bronze (#A6692E) provides a warm, analogous contrast for critical calls-to-action and active states.
- Headlines utilize 'Playfair' (serif) for a vintage, literary high-fashion feel, paired with 'PT Sans' (sans-serif) for functional, warm body text.
- A mobile-first, grid-based layout inspired by modern digital editorials, featuring generous negative space to let the photography take center stage.
- Elegant, thin-stroke monochrome icons with subtle gold accents when active, reinforcing the premium aesthetic of a wedding boutique.
- Smooth cinematic transitions, including fade-ins for image grids and subtle gold glows during upload progress events.
